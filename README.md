# Требования к окружению

Установленный node.js

# Установка зависимостей

```
npm i
```

# Запуск сервера

```
npm run start
```

# Взаимодействие с сервером

1. Заходим на [swagger.io](https://editor.swagger.io/)
2. Вставляем в поле ввода содержимое файла `docs.yaml`
3. Слева появляется список обрабатываемых серверов запросов
4. That's all
